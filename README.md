Python ile geliştirilen OOP tabanlı Kütüphane Otomasyon Sistemi. Kitap ve üye yönetimi, ödünç verme/iade işlemleri ve gecikme cezası hesaplama özellikleri içerir.
# 📚 Kütüphane Otomasyon Sistemi

Bu proje, Python ve Nesne Yönelimli Programlama (OOP) prensipleri kullanılarak geliştirilmiş bir Kütüphane Otomasyon Sistemidir.

## 🎯 Proje Amacı
Kütüphane içerisindeki kitap ve üye işlemlerini dijital ortamda yönetmek, ödünç verme ve iade süreçlerini düzenlemek, gecikme durumlarında otomatik ceza hesaplaması yapmaktır.

## ✨ Özellikler
- Kitap ekleme ve listeleme  
- Üye ekleme ve listeleme  
- Kitap ödünç verme sistemi  
- Kitap iade alma sistemi  
- Otomatik gecikme süresi hesaplama  
- Gecikmeye bağlı günlük 3 TL ceza sistemi  

## 🧠 Kullanılan Teknolojiler
- Python 3
- Nesne Yönelimli Programlama (OOP)
- JSON veri saklama
- `datetime` modülü

## 🏗️ Proje Yapısı
- Kitap sınıfı  
- Üye sınıfı  
- Kütüphaneci sınıfı  
- Ödünç işlem sınıfı  
- Kütüphane sistem yönetim sınıfı
